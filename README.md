# Описание проекта и инструкция по запуску

# ToDo лист API
### Описание
Благодаря этому проекту можно управлять задачами с использованием матрицы Эйзенхаура
### Технологии
Python 3.11
fastapi 0.119
### Запуск проекта в dev-режиме
- Установите и активируйте виртуальное окружение
- Установите зависимости из файла requirements.txt
```
pip install -r requirements.txt
```
- В папке с файлом main.py выполните команду:
```
unicorn main:app --reload
```

## Новые возможности v2.1.0

- Добавлена поддержка дедлайнов для задач
- Автоматический расчет срочности на основе дедлайна
- Вычисляемые поля: дни до дедлайна и статус срочности
- Новая статистика по срокам выполнения

## Матрица Эйзенхауэра

Квадранты рассчитываются автоматически:
- **Q1**: Важные и срочные (дедлайн ≤ 3 дня)
- **Q2**: Важные и несрочные (дедлайн > 3 дня или не указан)
- **Q3**: Неважные и срочные (дедлайн ≤ 3 дня)
- **Q4**: Неважные и несрочные (дедлайн > 3 дня или не указан)

## API Endpoints

### Tasks
- `GET /api/v2/tasks` - Получить все задачи
- `GET /api/v2/tasks/{id}` - Получить задачу по ID
- `POST /api/v2/tasks` - Создать новую задачу
- `PUT /api/v2/tasks/{id}` - Обновить задачу
- `DELETE /api/v2/tasks/{id}` - Удалить задачу
- `PATCH /api/v2/tasks/{id}/complete` - Отметить задачу выполненной
- `GET /api/v2/tasks/quadrant/{quadrant}` - Задачи по квадранту
- `GET /api/v2/tasks/status/{status}` - Задачи по статусу
- `GET /api/v2/tasks/search?q=...` - Поиск задач

### Statistics
- `GET /api/v2/stats/` - Общая статистика
- `GET /api/v2/stats/deadlines` - Статистика по дедлайнам
